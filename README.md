
## IDE Utilizada

O código foi desenvolvido na IDE Eclipse.

## Como Rodar o Código

1. Faça o download ou clone o repositório.
   ```sh
   git clone <URL_DO_REPOSITORIO>
Importe o projeto na IDE Eclipse.

Abra o Eclipse.
Vá para File > Import > Existing Projects into Workspace.
Selecione o diretório do repositório clonado.
Clique em Finish.
Navegue até a classe Main.

Descomente todas as funções que estão na classe ListarFuncionario (lf).

Execute a classe Main para testar o código.

Estrutura do Projeto
O projeto possui as seguintes classes principais:

Main: Classe principal que contém o método main para rodar o exemplo.
ListaFuncionario: Classe responsavel por execultar todos o metodos requisitados.
Pessoa: Classe que representa uma pessoa Nome e Data Nascimento. classe Pessoa é pai de classe funcionario.
Funcionario: Classe que representa um funcionário função e salário. classe Funcionario é filho de classe pessoa.
